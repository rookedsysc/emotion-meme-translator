# 프로젝트명: 희노애락 밈 통역사

## 1. 프로젝트 개요

<!-- ## 은 두 번째로 큰 제목(h2)을 나타냅니다 -->

### 1.1 목적
사용자의 상황을 입력받아 해당 상황에 적합한 밈을 변환하여 반환하는 서비스를 제공합니다.

### 1.2 배경
상황을 밈으로 표현함으로써 유쾌하고 효과적인 의사소통을 가능하게 합니다.

<!-- ### 은 세 번째로 큰 제목(h3)을 나타냅니다 -->

## 2. 주요 기능

- 텍스트 입력을 통한 상황 설명
- 상황에 맞는 밈 이미지 추천
- 밈 이미지 커스터마이징 옵션
- 생성된 밈 공유 기능
